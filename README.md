# Nostra – Responsive Fashion E-Commerce Website

## Description

Nostra is a responsive fashion e-commerce website built using **HTML, CSS, and JavaScript**. The website includes multiple pages such as **Home, Collections, and Contact Us**, designed to demonstrate modern frontend development concepts like responsive layouts, product cards, popup navigation, and interactive elements.

## Features

* Responsive design for **mobile and desktop devices**
* **Navigation bar with popup menu**
* **Product sections** including New Arrival and Most Wanted
* **Interactive heart icon** to like products
* **Sale tags and product cards**
* **Newsletter subscription section**
* **Contact form** for user messages
* Clean and simple **UI layout using CSS**

## Pages Included

* **Home Page** – Hero banner, brands section, product arrivals, offers, and newsletter
* **Collections Page** – Displays product collections
* **Contact Us Page** – Form with name, email, and message fields

## Technologies Used

* **HTML5** – Website structure
* **CSS3** – Styling and responsive layout
* **JavaScript** – Menu toggle, popup navigation, and UI interactions

## Responsive Design

The website is designed to work properly on **mobile, tablet, and desktop screens** using **CSS Flexbox and media queries**.

## How to Run the Project

1. Download or clone the project folder
2. Open the folder in your computer
3. Run **index.html** in any web browser
